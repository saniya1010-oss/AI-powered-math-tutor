### Overview:
Bingo is an adaptive math learning prototype built with **Streamlit**. It dynamically adjusts the difficulty of math problems based on a learner's performance using a simple **Logistic Regression ML Model**

### Installation & Setup
1. Clone the repository
2. ```python
   git clone https://github.com/<your-username>/AI-POWERED-MATH-TUTOR.git
   cd AI-POWERED-MATH-TUTOR
   ```
   Run the app preferably using VS code
   ```
   python -m streamlit run src/main.py
   ```

   ### Tech Stack

   **Frontend:** Streamlit
   **Backend:** Python
   **Model Used:** Logistic Regression
